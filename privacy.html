  • Улучшение качества AI-распознавания

3. Хранение данных

Ваши данные хранятся в защищённой базе данных Supabase с шифрованием. Фотографии обрабатываются через API OpenAI и не сохраняются на серверах после обработки.

Мы не продаём ваши данные. Данные могут передаваться:
